HẢO HÁN SMP LÀ GÌ?

Hảo Hán SMP là một server sinh tồn Minecraft cho nhiều người chơi được tạo ra bởi Ginn - server owner server được tạo ra sau khi anh đã chán 2y2c - một server không luật lệ với mục đích chill với những người bạn của anh!

Server có cấu hình như thế nào, có đủ để cân được server SMP?
Server hiện tại đang xài CPU : --Intel Xeon Platium 8124M @ 3.00Ghz--, với cpu đó thì đủ để cân được server --20TPS đều khi dưới hoặc bằng 15 players (Nếu có đủ ram có thể cân tầm 20 - > 25 players trong khoảng 19TPS) --. Ram của server là 8G RAM.
Server có backup gì không?
Server có auto backup mỗi 2 tiếng nên không lo mất world, đồ của người chơi.
Làm sao để vào server?
Trả lời các câu hỏi trong phần Sign Up rồi vào discord để xem IP.
Donate cho server sao?
Các bạn có thể liên hệ riêng cho Ginn#2005 bằng các phương thức MoMo, Paypal, Card.
